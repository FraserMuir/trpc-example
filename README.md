# Full stack monorepo prototype

This repo contains the setup for a full stack, typesafe , monorepo React / Node.js application using tRPC.

## Getting started

```sh
# Install dependencies
npm install

# Run dev server on port 3000
npm run dev
```

## Features

- [x] Typescript
- [x] React
- [x] Node.js
- [x] tRPC
- [x] Zod
- [ ] DrizzleORM
- [ ] Vitest
- [ ] Playwright
